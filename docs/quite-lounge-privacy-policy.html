QuietLounge 개인정보처리방침

시행일: 2026년 4월 7일

QuietLounge는 사용자의 개인정보를 수집하거나 외부 서버에 전송하지 않습니다.
모든 데이터는 사용자의 기기에만 저장됩니다.

1. 수집하는 개인정보

QuietLounge는 어떠한 개인정보도 수집하지 않습니다.

  • 로그인 정보를 수집하지 않습니다.
  • 사용자 활동을 추적하지 않습니다.
  • 분석, 광고, 마케팅 도구를 사용하지 않습니다.

2. 로컬에 저장되는 데이터

다음 데이터는 사용자의 브라우저 또는 기기 로컬 저장소에만 저장되며, 외부로 전송되지 않습니다.

  • 차단한 유저 목록 (personaId, 닉네임, 차단 일시)
  • 필터 모드 설정 (숨김/흐림)
  • 키워드 알림 설정 (채널, 키워드, 확인 주기)
  • 내 활동 통계 캐시

3. 네트워크 통신

QuietLounge는 다음 목적으로만 네이버 라운지 API와 통신합니다.

  • 게시글 작성자 식별 (personaId 매핑)
  • 내 활동 통계 조회
  • 키워드 알림을 위한 새 글 확인

이 통신은 사용자의 브라우저/앱에서 네이버 서버로 직접 이루어지며, QuietLounge의 자체 서버는 존재하지 않습니다. 어떤 데이터도 제3자에게 전송되지 않습니다.

4. 데이터 내보내기/가져오기

사용자가 직접 차단 목록을 JSON 파일로 내보내거나 가져올 수 있습니다. 이 파일은 사용자의 기기에 로컬로 저장되며, QuietLounge가 자동으로 업로드하거나 전송하지 않습니다.

5. 권한 사용 목적

  • storage: 차단 목록, 설정, 키워드 알림 데이터를 로컬에 저장
  • tabs: 차단된 게시글 수 뱃지 표시 및 콘텐츠 스크립트 통신
  • alarms: 키워드 알림 주기적 확인
  • notifications: 키워드 매칭 시 알림 표시 (선택적 권한)
  • 호스트 권한: lounge.naver.com 및 api.lounge.naver.com에서만 동작

6. 아동 개인정보

QuietLounge는 만 13세 미만의 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 변경 사항

본 개인정보처리방침이 변경될 경우, 이 페이지를 통해 공지합니다.
